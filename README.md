Sales Data Analysis Dashboard

Task Description: This project analyzes a sales dataset to identify monthly sales trends, best-selling products, and regional performance.

Tech Stack:
1. Python
2. Pandas
3. Matplotlib
4. Google Colab
   
Key Analysis Performed: 
* Data cleaning and handling missing values
* Monthly sales trend analysis
* Product-wise sales analysis
* Country-wise sales comparison
  
 Visualizations:
* Line chart for monthly sales
* Bar chart for product line sales
* Bar chart for top countries by sales

The analysis helps identify sales patterns, high-performing products, and regions contributing most to revenue.
